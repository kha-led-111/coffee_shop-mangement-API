# coffee_shop-mangement-API
A full-featured REST API built with .NET Core 8 for managing a coffee shop — designed to integrate seamlessly with a Flutter mobile application.
✨ Features

🔐 JWT Authentication — Role-based access (Admin, Cashier, Barista)<br>
📋 Menu Management — Categories and items with image upload<br>
🛒 Order Management — Create, track, and update orders<br>
🪑 Table & Reservations — Manage tables and customer reservations<br>
⚡ Real-time Updates — Live order status via SignalR WebSockets<br>
🖼️ File Upload — Multipart image upload (jpg, png, webp — max 5MB)<br>
📄 Swagger UI — Interactive API documentation at /swagger<br>
🌍 Flutter-ready — Consistent JSON response wrapper on every endpoint<br>

Prerequisites<br>
   .NET 8 SDK<br>
   SQL Server (local or remote)<br>
   Visual Studio 2022 or VS Code
