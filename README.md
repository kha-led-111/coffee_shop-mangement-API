# coffee_shop-mangement-API
A full-featured REST API built with .NET Core 8 for managing a coffee shop — designed to integrate seamlessly with a Flutter mobile application.
✨ Features

🔐 JWT Authentication — Role-based access (Admin, Cashier, Barista)
📋 Menu Management — Categories and items with image upload
🛒 Order Management — Create, track, and update orders
🪑 Table & Reservations — Manage tables and customer reservations
⚡ Real-time Updates — Live order status via SignalR WebSockets
🖼️ File Upload — Multipart image upload (jpg, png, webp — max 5MB)
📄 Swagger UI — Interactive API documentation at /swagger
🌍 Flutter-ready — Consistent JSON response wrapper on every endpoint

Prerequisites
   .NET 8 SDK
   SQL Server (local or remote)
   Visual Studio 2022 or VS Code
